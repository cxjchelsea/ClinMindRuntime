# 六、判断文档是否应该新增的规则

新增文档前先问：

```text
1. 它是否服务当前或下一 Phase？
2. 它是否能转化为实现规格或任务清单？
3. 它是否补了总设计里的明确缺口？
4. 它是否会让 AI 编码工具更安全地执行？
5. 它是否会与现有文档重复？
```

如果答案大多是否，则不要新增。
